concepto general

Una aplicacion es un tipo de programa que le permite realizar una tarea concreta. Puede utilizar aplicaciones para realizar una de las siguientes tareas: crear documentos de texto, como cartas o informes, trabajar con hojas de calculo, escuchar musica, explorar internet, crear, editar o ver imagenes y videos.

editor de texto

permite leer, crear o modificar un texto simple sin ninguon formato.

diccionario

permite buscar definiciones de una palabra.

visor de imagenes

permite visualizar archivos de una imagen o recopilaciones de imagenes.

mapa de caracteres

permite seleccionar las letras y los simbolos del juego de caracter Unicode y pegarlos en cualquier aplicacion.

gestor de archivos

permite visualizar carpetas y sus contenidos.Puede usar esta aplicacion para copiar, mover y clasificar sus archivos. La aplicacion tambien permite acceder a CD, unidades flash USB y otros medios extraibles.

terminal

permite acceder a la linea de comandos del sistema
